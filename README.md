# python-cli-poetry-api-flask-sqlite-connect

## Description
Connect to sqlite file for a flask project.

## Tech stack
- python
	- poetry
- sqlite

## Docker stack
- python:latest

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
